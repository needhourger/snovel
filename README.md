
## Usage
1. 程序默认启动会以当前目录为启动路径,并扫描当前目录下的txt文件,载入书架
   ```
   python snoval.py
   ```
2. 使用tab切换焦点,使用方向键选择图书,章节
   
3. 使用Ctrl+x载入指定图书章节
   
4. 使用ctrl+c退出程序
